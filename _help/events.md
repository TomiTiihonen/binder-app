Events capture moments — a concert, a dinner, a trip, a birthday. Unlike calendar events, they're a record of what happened, with room for photos, location, and your own notes.

## Creating an event

1. Tap **+** and choose **Event**.
2. Add a title and description.
3. Optionally set a location, cover photo, and gallery photos.

## Rich event fields

Events support the full set of attachments:

- **Cover photo** — the one image that represents the event
- **Gallery photos** — additional photos from the event
- **Location** — pick a place from the map or search
- **Reminder** — nudge yourself to revisit the memory

See [Photos and galleries](/help/photos-and-galleries/) and [Locations and maps](/help/locations-and-maps/) for details.

## Where events shine

- Concerts, gigs, meals, weekend trips
- Milestones — first day at a new job, a visit with family
- Anything you want to remember with context: *when*, *where*, and *what it looked like*
