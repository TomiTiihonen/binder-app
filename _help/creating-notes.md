Notes are the simplest entry type in Binder — quick text captures with optional reminders.

## Creating a note

1. Tap the **+** button in the toolbar.
2. Choose **Note** from the entry types.
3. Start typing. Changes are saved automatically.

## Tips

- Use headings to organise long notes
- Pin important notes to the top of your journal
- Add a reminder to revisit a note later

{% include callout.html type="tip" body="On macOS, notes save automatically as you type — no save button needed." %}
