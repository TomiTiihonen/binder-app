Notes are Binder's most flexible entry type — free-form text for anything that doesn't fit another category. Use them for journaling, drafts, ideas, meeting notes, or lists of thoughts that don't need structure.

## Creating a note

1. Tap **+** and choose **Note**.
2. Give it a title.
3. Start typing in the body. Changes save automatically.

## Formatting

Notes support lightweight inline formatting — headings, bullets, quotes, and code blocks. On iOS, the formatting styles stick across autocorrect replacements.

## Reminders

Any note can have a reminder. See [Reminders](/help/reminders/) for details.

## Platform differences

- **macOS**: fully inline editing — there's no separate edit mode, just type.
- **iOS**: tap a note to view it, tap again to edit in a sheet.
