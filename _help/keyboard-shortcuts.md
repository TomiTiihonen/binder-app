Binder on macOS is built for the keyboard. Here are the shortcuts that ship in the app.

## Global

| Shortcut | Action |
|---|---|
| **⌘N** | New Entry |
| **⇧⌘O** | Search Entries |
| **⌘P** | Print current entry |
| **⌘?** | Open Binder Help |
| **Shift Shift** (double-tap) | Open the command palette |

## Find in view

When viewing an entry's detail pane:

| Shortcut | Action |
|---|---|
| **⌘F** | Open Find bar for the current entry |
| **⌘G** | Jump to next match |
| **⇧⌘G** | Jump to previous match |
| **Esc** | Close the Find bar |

## Command palette

With the palette open:

| Shortcut | Action |
|---|---|
| **↑ / ↓** | Move between results |
| **Enter** | Open the selected result (or run AI search if nothing matched) |
| **Esc** | Dismiss |

## Quick note window

| Shortcut | Action |
|---|---|
| **⌘Enter** | Save and close |
| **Esc** | Dismiss without saving |
