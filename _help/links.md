Link entries save URLs with rich previews. When you paste a link, Binder fetches the page's Open Graph metadata — title, description, image, and site name — and shows a card preview.

## Saving a link

1. Tap **+** and choose **Link**.
2. Paste or type a URL.
3. The preview card populates automatically.
4. Add your own title, description, or personal annotation.

## The unfurl card

The preview is a real-time unfurl of the page: title, description, and site image if available. If a site blocks metadata fetching, you can still save the link with your own title and notes.

## Annotations

Link entries have a personal annotation field separate from the fetched description. Use it for why *you* saved the link — a quote, a reminder to read it, what to share it with.

## Reminders

Set a reminder on a link to read or revisit it later. See [Reminders](/help/reminders/).
